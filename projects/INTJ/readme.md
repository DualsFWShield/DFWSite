# Everything on INTJ

Un site web complet sur le type de personnalité INTJ (Architecte) : explications, exemples, ressources, FAQ, et plus.

## Fonctionnalités

- Portrait détaillé de l'INTJ
- Exemples de personnages (films, séries, entrepreneurs, scientifiques)
- FAQ interactive
- Glossaire MBTI
- Thème clair/sombre (switch)
- Menu responsive avec dropdowns

## Installation

1. Clonez ce dépôt ou copiez les fichiers dans votre serveur local.
2. Ouvrez `index.html` dans votre navigateur.

## Structure

- `index.html` : page d'accueil
- `css/style.css` : styles principaux
- `css/light-mode.css` et `css/dark-mode.css` : thèmes
- `js/main.js` : menu, dropdowns, interactions
- `js/theme-switcher.js` : gestion du thème clair/sombre
- Autres fichiers HTML : pages de contenu

## Dépendances

- [Font Awesome](https://fontawesome.com/) (CDN)
- [Google Fonts](https://fonts.google.com/) (CDN)

## Personnalisation

Modifiez les fichiers HTML/CSS/JS selon vos besoins pour ajouter du contenu ou adapter le style.
